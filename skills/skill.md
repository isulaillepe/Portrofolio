# Skill: `format_luxury_case_study`

**Trigger:** Use this skill whenever generating a section for a specific project (e.g., CourtConnect, Flyer Tool).

**Behavior:**
Instead of listing project details as raw text, format them using the "Luxury Case Study" structure:

1.  **The Hook:** A one-sentence, high-impact summary of the project's business or technical value.
2.  **The Stack:** A beautifully spaced row of minimalist icons or text chips (e.g., `[ Flutter ] [ Node.js ] [ MongoDB ]`).
3.  **The Architecture / Leadership:** A brief, 2-line explanation of the technical architecture or the leadership role taken (e.g., "Led a 6-member team...").
4.  **The Impact:** The measurable result (e.g., "Eliminated manual design tasks," "Advanced to 2nd round of PitchArena").

**Output Constraint:** Ensure all HTML/CSS generated using this skill applies the "Dark Charcoal" and "Cyber Sapphire" color palette defined in AGENTS.md.
